Sobre o projeto

O EcoData é um repositório automatizado de séries econômicas, financeiras e fiscais voltado à análise de conjuntura. As bases são atualizadas por scripts em Python e disponibilizadas em formatos abertos para uso em planilhas, softwares estatísticos, notebooks e relatórios.

Frequência principal
Mensal
Fontes iniciais
Yahoo + BCB/SGS
Formatos disponíveis
CSV, XLSX, JSON, Parquet
Observação: esta é uma versão inicial do projeto. Novas fontes, como IBGE/SIDRA, Tesouro Nacional e Ipeadata, podem ser incorporadas posteriormente.

Downloads por série individual

Além das bases completas, cada variável da base consolidada mensal também é disponibilizada separadamente. Cada arquivo individual contém duas colunas: data e valor.

As séries individuais são geradas automaticamente a partir da base consolidada mensal. Atualmente, cada série é disponibilizada em CSV e XLSX.

Metodologia de atualização

As séries são coletadas automaticamente por scripts em Python. Cada fonte possui um script próprio de coleta, tratamento e exportação. Em seguida, as bases mensais são consolidadas em uma base única.

Etapa Arquivo Função
Coleta Yahoo Finance src/coleta_yahoo.py Baixa dados de mercado financeiro e converte para frequência mensal.
Coleta BCB/SGS src/coleta_bcb_sgs.py Baixa séries macroeconômicas, monetárias, cambiais e de crédito.
Consolidação da base src/consolidar_base.py Une as bases mensais em uma base consolidada.
Dicionário consolidado src/consolidar_dicionario.py Junta os dicionários de variáveis das diferentes fontes.
Metadados do site src/gerar_metadados_site.py Gera arquivos auxiliares de documentação e atualização.
Pipeline geral src/atualizar_base.py Executa todas as etapas do projeto.

A automação pode ser executada manualmente ou em agenda programada pelo GitHub Actions.

Como usar no Python

Exemplo simples para carregar a base consolidada mensal: 

import pandas as pd

url = "data/final/consolidada/base_conjuntura_mensal.csv"

df = pd.read_csv(url)
df["data"] = pd.to_datetime(df["data"])

print(df.head())

Para uso fora do GitHub Pages, basta baixar o arquivo CSV, XLSX, JSON ou Parquet diretamente pelos botões acima.
